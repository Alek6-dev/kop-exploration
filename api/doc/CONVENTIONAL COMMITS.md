# Conventional commits

Pour des messages de commits lisible par les humains et les machines (https://www.conventionalcommits.org/en/v1.0.0/)

### Pour facilité son utilisation

- Le projet embarque ce module NPM : https://github.com/commitizen/cz-cli

- Un hook GIT : config/git/hooks/commit-msg

- Plugin PHPStorm : 

  - https://plugins.jetbrains.com/plugin/9861-git-commit-template
  - https://plugins.jetbrains.com/plugin/13389-conventional-commit

- Plugin VSCode :
  - https://marketplace.visualstudio.com/items?itemName=vivaxy.vscode-conventional-commits
